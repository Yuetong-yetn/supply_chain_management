# Example Data

本目录包含系统演示、初始化和测试所需的全部假数据。

- categories.json: 商品类别及父子类别
- products.json: 商品主数据
- suppliers.json: 供应商主数据
- supplier_products.json: 供应商商品关系
- warehouses.json: 仓库主数据
- stores.json: 门店主数据
- users.json: 基础用户，导入时密码会自动哈希
- inventory_seed.json: 初始库存
- monthly_sales_facts.json: 最近12个月销售事实
- purchase_orders.json: 采购订单及明细
- inbound_orders.json: 入库单及明细
- outbound_orders.json: 出库单及明细
- replenishment_requests.json: 补货申请
- stock_transactions.json: 示例库存流水
- ai_recommendations.json: 初始AI补货建议
- supplier_score_snapshots.json: 供应商评分快照
- promotions.json: 促销活动数据
