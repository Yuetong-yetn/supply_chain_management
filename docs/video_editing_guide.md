# 视频剪辑落地说明

现有素材：

- 原视频：`屏幕录制 2026-07-01 020909.mp4`
- 字幕文件：`docs/video_subtitles.srt`
- 配音讲稿：`docs/video_voiceover_script.md`
- 语音生成脚本：`scripts/generate_video_voiceover.ps1`

## 剪映或 Premiere 的最快做法

1. 导入 `屏幕录制 2026-07-01 020909.mp4`。
2. 直接导入 `docs/video_subtitles.srt`，让软件自动挂载时间轴字幕。
3. 配音有两种方案：
   - 真人录音：照着 `docs/video_voiceover_script.md` 读一遍。
   - 机器配音：先运行 `scripts/generate_video_voiceover.ps1`，把生成的 `wav` 片段拖入时间轴。
4. 片头建议加上项目名：`供应链协同与智能补货管理系统`。
5. 片尾建议保留一句总结：`项目展示了采购、库存、补货、出库、签收到分析的完整闭环。`

## 字幕样式建议

- 字号：中号偏大，保证教室后排也能看清。
- 颜色：白字加半透明黑底。
- 位置：底部居中，避免遮住左侧导航和图表。

## 录答辩版的建议

- 讲解重点优先放在业务闭环，不要过度展开代码细节。
- 如果时间不够，可以重点保留四段：
  首页看板、采购入库、门店补货闭环、AI 补货建议。
- 如果老师问数据库，补充说明：
  项目优先使用 OceanBase，连接失败时自动回退 SQLite，便于本地演示。
